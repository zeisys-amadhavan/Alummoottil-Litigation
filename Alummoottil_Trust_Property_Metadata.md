# Alummoottil Tharavad Trust — Property Metadata

> **Organising principle:** the same physical asset is recorded in THREE independent registers, maintained by THREE different authorities. They are not the same thing and they can disagree:
>
> - **A. LAND / SURVEY RECORDS** — Revenue Dept (ReLIS / Thandapper / Village Office). Records the *land parcel*: survey number, area, registered landholder, basic land tax.
> - **B. SETTLEMENT / TRANSACTION RECORDS** — Registration Dept (Cheppad SRO). Records *transactions*: who conveyed what share, when, by which deed. A registered deed is a transaction; it does NOT change the land or building register until mutation is separately applied for and effected.
> - **C. BUILDING RECORDS** — LSGD (Sanchaya / Cheppad Grama Panchayath). Records the *structure*: door number, plinth, building tax, and the name the building tax stands in.
>
> Section D reconciles the three and flags discrepancies (these are evidentiarily important). Fields marked **[TO COLLECT]** are pending.
> This file covers TRUST property only. Individuals' personal property is in a separate register.
> Last updated: 2026-06-26
